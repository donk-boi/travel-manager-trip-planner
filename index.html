Travel Manager

Trip Planner Android Mobile Application

About the App

Travel Manager - Trip Planner is an Android mobile application designed for independent travelers who want to organize vacation and excursion information in one place. The app allows users to add, edit, delete, search, and share vacation records, manage associated excursions, set travel alerts, and generate itinerary reports.

Installation Instructions

To install the application, download the APK file and install it on an Android device or Android emulator. Android may require permission to install apps from unknown sources before the APK can be installed.

For emulator testing, the APK can also be installed using Android Debug Bridge with the following command:

adb install TravelManager-release.apk

Application Features

The application includes vacation management, excursion management, validation, search functionality, itinerary report generation, scheduled alerts, sharing functionality, local Room/SQLite database storage, reusable object-oriented code, input sanitization, and a user-friendly Android interface.
